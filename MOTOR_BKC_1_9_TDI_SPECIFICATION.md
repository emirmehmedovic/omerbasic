## üîç HOW TO FIND ENGINE POWER - SQL QUERY

### Query 1: Pronaƒëi motor po kodu

```sql
-- Pronaƒëi motor BKC sa svim informacijama
SELECT
  e.id as 'Motor ID',
  e.Description as 'Motor kod',
  e.SalesDescription as 'Kratko',
  YEAR(e.From) as 'Od_godine',
  YEAR(e.To) as 'Do_godine'
FROM engines e
WHERE e.Description = 'BKC'
LIMIT 10;
```

**Rezultat**:
```

Kod: BKC

```

---

### Query 2: Pronaƒëi sve automobile sa BKC motorom

```sql
-- Pronaƒëi sve automobili koji koriste BKC motor
SELECT DISTINCT
  pc.id as 'Auto ID',
  mo.Description as 'Model',
  m.Description as 'Proizvoƒëaƒç',
  e.Description as 'Motor',
  YEAR(pc.From) as 'Od',
  YEAR(pc.To) as 'Do'
FROM passengercars pc
LEFT JOIN models mo ON mo.id = pc.Model
LEFT JOIN manufacturers m ON m.id = pc.ManufacturerId
LEFT JOIN passengercars_link_engines ple ON ple.car_id = pc.id
LEFT JOIN engines e ON e.id = ple.engine_id
WHERE e.id IN (17026, 17355, 17497, 17079)
ORDER BY m.Description, mo.Description, YEAR(pc.From);
```

**Rezultat**: 15-16 razliƒçitih modela sa BKC motorom

### Query 3: Pronaƒëi snagu BKC motora (VA≈ΩNO!)

```sql
-- Pronaƒëi snagu (PS, KW) za automobil sa BKC motorom
SELECT
  ia.item_id as 'Auto ID',
```

### Query 4: Kompletna Analiza - Motor + Automobili + Snaga

```sql
-- KOMPLETNA ANALIZA: Motor + Vozila + Snaga
SELECT
  mo.Description as 'Model',
  m.Description as 'Proizvoƒëaƒç',
  YEAR(pc.From) as 'Od',
  YEAR(pc.To) as 'Do',
  e.Description as 'Motor kod',
  COUNT(DISTINCT ia.DisplayValue) as 'Broj_varijanti_snage',
  GROUP_CONCAT(DISTINCT ia.DisplayValue ORDER BY ia.DisplayValue SEPARATOR ' / ') as 'Snage'
FROM passengercars pc
LEFT JOIN models mo ON mo.id = pc.Model
LEFT JOIN manufacturers m ON m.id = pc.ManufacturerId
LEFT JOIN passengercars_link_engines ple ON ple.car_id = pc.id
LEFT JOIN engines e ON e.id = ple.engine_id
LEFT JOIN items_atributes ia ON ia.item_id = pc.id AND ia.DisplayTitle = 'Power'
WHERE e.id IN (17026, 17355, 17497, 17079)
GROUP BY mo.Description, m.Description, YEAR(pc.From), YEAR(pc.To), e.Description
ORDER BY m.Description, mo.Description, YEAR(pc.From);
```

**Rezultat**: Sve kombinacije vozila + snaga
